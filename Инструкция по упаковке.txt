Краткая инструкция такова:
1. Правая кнопка мыши на имени проекта - "Опубликовать..."
2. В открывшемся мастере в качестве "Целевого объекта" выбрать "Папка" - "Далее"
3. Снова "Папка" - "Далее"
4. "Расположение папки" - оставляем значение по умолчанию (bin\Release\net6.0-windows\publish\) - "Готово" - "Закрыть"
5. В появившемся окне публикации внизу жмём "Все параметры". Появится окно редактирования профиля.
6. Оставляем всё по умолчанию, кроме параметра "Целевая среда выполнения" - там ставим "Win-x64". При этом внизу появится раздел "Параметры публикации файлов"
7. В "Параметрах публикации файлов" ставим галочку "Создать отдельный файл". Сохраняемся
8. Теперь в Обозревателе решений из папки "Properties" - "PublishProfiles" открываем файл FolderProfile.pubxml
9. На запрос о несовместимости строк с предложение исправить говорим "НЕТ"!
10. В конец раздела "PropertyGroup" (после <PublishReadyToRun>false</PublishReadyToRun>) вставляем строчку <IncludeNativeLibrariesForSelfExtract>true</IncludeNativeLibrariesForSelfExtract>
11. После этой строчки вставляем строку <DebugType>embedded</DebugType>
12. Сохраняем файл FolderProfile.pubxml, закрываем его и жмём кнопку "Опубликовать"
13. По окончании публикации идём в папку, указанную на шаге 4. Там в папке "win-x64" должен лежать ЕДИНСТВЕННЫЙ файл приложения
